# Mip-NeRF 复现

**题目**：请使用 PyTorch 实现 Mip-NeRF 的基本算法，要求代码能够在 nerf_synthetic 数据集上运行。

**要求**：

- 生成内容至少包括以下 5 个文件或文件夹：
  - `README.md`
  - `train.py`
  - `eval.py`
  - `models/models.py`
  - `datasets/datasets.py`
- 在`README.md`文件中需要写明如何配置运行环境以及运行代码的指令，能够正常得到运行结果；

**数据**：

在子文件夹`nerf_synthetic/`中提供了官方的 nerf_synthetic 数据集。
